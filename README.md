# Projeto-3-Catalogo

## Descrição

Projeto elaborado com finalidade acadêmica, para aplicação de conceitos e tecnologias aprendidas no curso FullStack da BlueTech, no ano de 2022 no módulo - 2.

### Stacks

Módulos utilizados:
*JavaScript
*Express
*Nodemon
*PgAdmin 4
Foram criadas rotas e renderizado os arquivos nas páginas. Utilização de um banco de dados real SQL, através do PgAdmin 4.
